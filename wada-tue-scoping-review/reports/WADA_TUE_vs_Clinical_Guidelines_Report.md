# WADA TUE制度と最新診療ガイドラインの乖離に関する調査レポート

**作成日: 2026年3月27日**

---

## エグゼクティブサマリー

世界アンチ・ドーピング機構（WADA）の治療使用特例（TUE: Therapeutic Use Exemption）制度は、アスリートが禁止物質を治療目的で使用するための重要な枠組みである。しかし、臨床医学の急速な進歩により、最新の診療ガイドラインで推奨される第一選択薬が WADA 禁止表に抵触する、あるいは TUE ガイドラインが最新のエビデンスを十分に反映していないケースが複数の疾患領域で生じている。

本レポートでは、**喘息、ADHD、2型糖尿病、男性性腺機能低下症、糖質コルチコイド使用、GLP-1受容体作動薬、心血管疾患、PCOS**の8領域について、WADA TUE制度と最新診療ガイドラインの間に存在するギャップを体系的に整理する。

## 1. 喘息（Beta-2作動薬・吸入ステロイド）

### WADAの現行規定（2026年禁止表）

| 物質 | ステータス |
|---|---|
| **全てのβ2作動薬** | **常時禁止**（S3） |
| 吸入サルブタモール | 例外許可：24時間で1600μg以下（8時間で600μg以下） |
| 吸入ホルモテロール（formoterol） | 例外許可：24時間で54μg以下（delivered dose、12時間で36μg以下） |
| 吸入サルメテロール | 例外許可：24時間で200μg以下（**2026年改訂: 8時間で100μg以下**） |
| 吸入ビランテロール | 例外許可：24時間で25μg以下 |
| 上記以外のβ2作動薬（経口・注射） | **TUE必須** |
| 糖質コルチコイド（経口・注射） | **競技会時禁止**（S9）、TUE必須 |

### GINA 2025ガイドラインの推奨

GINA（Global Initiative for Asthma）2025は、喘息管理における**パラダイムシフト**を確立した：

- **Track 1（推奨）**: 全ステップで **ICS-ホルモテロール（ICS-formoterol）をas-needed relieverとして使用**。ステップ3-5では維持療法（MART: Maintenance and Reliever Therapy）としても使用。
- **Track 2（代替）**: SABA（短時間作用型β2作動薬）をrelieverとして使用 + 別個のICS含有コントローラー。
- **重要**: SABA単独療法（サルブタモール吸入のみ）は**いかなるステップでも推奨されない**。

### ギャップの詳細

| 項目 | ギャップの内容 | リスク評価 |
|---|---|---|
| **ホルモテロールの用量上限** | GINA Track 1のMARTレジメンでは、維持量+急性増悪時のas-needed使用で1日のホルモテロール総量がWADA上限（54μg delivered dose/24h）に近づく可能性がある。特に増悪期は臨床的に必要な用量がWADA閾値を超えるリスクがある | **中〜高** |
| **尿中ホルモテロール濃度閾値** | WADA閾値（尿中ホルモテロール40ng/mL超で陽性）は、MART使用時の高用量期に超過する可能性がある。アスリートは治療的用量であっても陽性判定を受けるリスクがある | **高** |
| **SABA単独使用の推奨廃止** | WADAは吸入サルブタモールを閾値以下であればTUE不要で許可しているため、アスリートがSABA単独で管理するインセンティブが残る。しかしGINAはSABA単独を明確に非推奨としており、臨床的に不適切な治療を選択するジレンマが生じる | **中** |
| **重症喘息における全身性GC** | 急性増悪時の短期経口プレドニゾロン投与は標準治療だが、競技会時には禁止。ウォッシュアウト期間（経口GCで3〜10日）が大会スケジュールと合わない場合、TUE申請が間に合わないリスク | **高** |
| **生物学的製剤（biologics）の未記載** | 重症喘息に対するオマリズマブ、メポリズマブ、デュピルマブ等は禁止表に記載なく使用可能だが、TUEガイドラインでの位置づけが不明確 | **低** |

### 具体的シナリオ

> **例**: 冬季スポーツ選手（クロスカントリースキー）がEIB（運動誘発性気管支攣縮）と診断。GINA Track 1に基づきブデソニド/ホルモテロール（200/6μg）のMARTが処方される。維持量として1日2回+症状時にas-needed使用。トレーニング強度が高い冬場、1日の総ホルモテロール使用量が48μg（delivered dose）に達し、WADAの54μg上限に近接。レース当日の追加使用で閾値超過のリスクが生じる。

## 2. ADHD（注意欠如多動症）

### WADAの現行規定

| 物質 | ステータス |
|---|---|
| アンフェタミン類（アデロール等） | **競技会時禁止**（S6.A: 非特定刺激薬） |
| メチルフェニデート（リタリン、コンサータ等） | **競技会時禁止**（S6.B: 特定刺激薬） |
| リスデキサンフェタミン（ビバンセ/Vyvanse） | **競技会時禁止**（S6: 刺激薬） |
| アトモキセチン（ストラテラ） | 禁止対象外 |
| グアンファシン | 禁止対象外（**2025年に明確化**） |

### 最新診療ガイドライン（2022年オーストラリアガイドライン、NICE 2024、APA等）

- **第一選択薬**: 刺激薬（メチルフェニデートまたはリスデキサンフェタミン/アンフェタミン類）
  - 成人ADHD：刺激薬が最もエビデンスレベルが高く、効果量も最大
  - 各国ガイドラインで一貫して刺激薬を第一選択として推奨
- **第二選択薬**: アトモキセチン、グアンファシン（効果発現まで数週間を要する）

### ギャップの詳細

| 項目 | ギャップの内容 | リスク評価 |
|---|---|---|
| **第一選択薬が全て禁止** | 全主要ガイドラインが推奨する第一選択薬（メチルフェニデート、リスデキサンフェタミン、アンフェタミン）は全てWADA禁止表に記載。アスリートは最適治療のためにTUE申請が必須 | **高** |
| **TUE申請の煩雑さ** | WADA TUEガイドライン（Version 8.0, 2026年1月）は、詳細な心理学的・神経心理学的評価、DSM-5に基づく小児期からの症状の文書化、成人診断の場合の包括的評価を要求。費用・時間的負担が大きい | **高** |
| **競技会時のみ禁止の矛盾** | 刺激薬は競技会時のみ禁止（out-of-competition では許可）。しかしADHDは24時間365日の障害であり、競技日だけ服薬を中断することは臨床的に不適切。長時間作用型製剤（コンサータ等）の突然の中断は離脱症状や認知機能の急激な悪化を招く | **高** |
| **成人ADHD診断の増加との乖離** | 近年、成人ADHD診断が世界的に増加。WADA TUEガイドラインは成人期初回診断に対してより厳格な審査を求めるが、臨床的には成人期診断は確立された医学的事実 | **中** |
| **非刺激薬の効果不十分性** | TUE基準の一つに「禁止されていない代替治療がないこと」があるが、アトモキセチンやグアンファシンは約30-40%の患者で効果不十分。「まず非刺激薬を試すべき」とされる場合、最適治療へのアクセスが遅れる | **中** |

### 具体的シナリオ

> **例**: 25歳プロサッカー選手が成人期にADHDと診断。精神科医はリスデキサンフェタミン（Vyvanse）70mgを処方。リーグ戦シーズン中（ほぼ毎週末が「競技会時」）は事実上使用できない。アトモキセチンに切替えるが効果不十分で集中力低下・チームパフォーマンスに影響。TUE申請には小児期の記録、心理検査等が必要だが、海外移籍歴があり過去の医療記録収集に困難を伴う。

## 3. 2型糖尿病とGLP-1受容体作動薬

### WADAの現行規定

| 物質 | ステータス |
|---|---|
| インスリン | **常時禁止**（S4: ホルモン・代謝調節薬）、TUE必須 |
| GLP-1受容体作動薬（セマグルチド、リラグルチド、チルゼパチド等） | **現在は禁止対象外** |
| セマグルチド・チルゼパチド | **2024年よりモニタリングプログラムに収載**（2026年も継続監視中） |
| SGLT2阻害薬（エンパグリフロジン等） | 禁止対象外 |
| メトホルミン | 禁止対象外 |

### ADA（米国糖尿病学会）Standards of Care 2025の推奨

2025年版ADAガイドラインでは、2型糖尿病の薬物治療アルゴリズムが大幅に改訂された：

- **心血管リスクが高い患者**: メトホルミンに加え、**GLP-1受容体作動薬**またはSGLT2阻害薬を早期から推奨
- **体重管理の重視**: GLP-1受容体作動薬の減量効果を積極的に活用する方針
- **腎保護**: SGLT2阻害薬とGLP-1受容体作動薬の腎保護効果を評価
- **インスリン**: 依然として1型糖尿病では必須、2型でも進行例で必要だが、可能な限りGLP-1受容体作動薬を先行して使用する流れ

### ギャップの詳細

| 項目 | ギャップの内容 | リスク評価 |
|---|---|---|
| **インスリンTUEの負担** | 1型糖尿病アスリートにとってインスリンは生命維持に不可欠だが、TUE申請が必須。手続きは比較的定型化されているが、毎年の更新負担がある | **中** |
| **GLP-1受容体作動薬の将来的禁止リスク** | セマグルチド・チルゼパチドは現在禁止対象外だが、2024年からモニタリングプログラムに収載。体重減少効果が体重階級制スポーツで有利に働く可能性があり、将来的に禁止される可能性がある。しかしADA 2025は2型糖尿病の標準治療としてGLP-1受容体作動薬を強く推奨しており、禁止された場合、糖尿病アスリートの最適治療へのアクセスが制限される | **高（将来的）** |
| **体重階級制スポーツとの衝突** | 柔道、レスリング、ボクシング等の体重階級制スポーツにおいて、GLP-1受容体作動薬の使用は体重管理と治療の境界が曖昧になる。WADAは「新たな体重管理薬カテゴリー」の創設を検討中 | **高（将来的）** |
| **TUEガイドラインの遅延** | WADA糖尿病TUEガイドライン（Version 5.1, 2023年10月）はインスリンのみを対象禁止物質として記載。GLP-1受容体作動薬が将来禁止された場合のTUE基準は未策定 | **中** |
| **2型糖尿病の複合治療** | ADA 2025ではGLP-1受容体作動薬+SGLT2阻害薬+メトホルミンの3剤併用を推奨するケースがあるが、GLP-1が禁止された場合、この標準治療レジメンが崩壊する | **高（将来的）** |

### 具体的シナリオ

> **例**: 35歳の2型糖尿病を持つマスターズ水泳選手。ADAガイドラインに基づきメトホルミン+セマグルチド（Ozempic）で良好な血糖コントロールを達成中。将来セマグルチドが禁止表に追加された場合、治療変更を迫られる。インスリンへの変更はTUEが必要なうえ、低血糖リスクが増大し、水泳中の危険が生じる。

## 4. 男性性腺機能低下症（テストステロン補充療法）

### WADAの現行規定

| 物質 | ステータス |
|---|---|
| テストステロン（全製剤） | **常時禁止**（S1: 蛋白同化薬） |
| hCG（ヒト絨毛性ゴナドトロピン） | **常時禁止**（S2） |
| クロミフェン | **常時禁止**（S4: ホルモン・代謝調節薬） |

### WADA TUEガイドラインの制限

WADA TUEガイドライン（Male Hypogonadism）は以下の厳格な制限を設けている：

> **「TUEは器質的（organic）病因の性腺機能低下症にのみ承認されるべきである。機能的（functional）障害による男性ホルモン欠乏に対するTUEは承認されるべきではない。」**

**器質的病因の例（TUE承認対象）:**
- クラインフェルター症候群
- 先天性無精巣症/停留精巣
- カルマン症候群
- 下垂体腫瘍
- 外傷性脳損傷による下垂体機能障害
- 精巣への放射線照射/化学療法後

**機能的病因の例（TUE非承認）:**
- 加齢に伴うテストステロン低下（LOH: Late-Onset Hypogonadism）
- 肥満関連の低テストステロン
- オピオイド誘発性性腺機能低下症
- 過度のトレーニングによる視床下部性機能低下

### 最新臨床ガイドライン（Endocrine Society 2018、EAU 2024）

- **Endocrine Society 2018ガイドライン**: 症状と一貫して低いテストステロン値を示す男性にテストステロン補充療法（TRT）を推奨。器質的・機能的の区別は治療適応の絶対的基準ではなく、**症状の存在と生化学的確認**が治療開始の根拠。
- **EAU 2024ガイドライン**: Late-Onset Hypogonadism（LOH）に対しても、生活習慣改善で改善しない場合のTRT適応を認めている。
- **TRAVERSE試験（2023年）**: 45-80歳の性腺機能低下症男性におけるTRTの心血管安全性を実証し、TRT適用拡大の根拠を提供。

### ギャップの詳細

| 項目 | ギャップの内容 | リスク評価 |
|---|---|---|
| **機能的性腺機能低下症の除外** | WADAは機能的性腺機能低下症へのTUEを明確に否定。しかし内分泌学会ガイドラインは、可逆的原因への対処後も改善しない場合のTRTを認めている。マスターズアスリートにおけるLOHが最も影響を受ける | **非常に高** |
| **マスターズアスリートの増加** | 40歳以上のマスターズアスリート人口は増加の一途。加齢に伴うテストステロン低下は生理的現象だが、症状が顕著な場合の臨床的TRT適応とWADA規定が正面から衝突する | **非常に高** |
| **診断閾値の不一致** | Endocrine Societyはテストステロン値<300ng/dL（朝空腹時、2回測定）を診断閾値とするが、WADAは特定の数値閾値を設定せず器質的病因の証明を要求。同じ生化学的所見でも、「器質的原因の証明」の有無で治療アクセスが異なる | **高** |
| **過去のAAS使用による性腺機能低下** | 元アスリートが過去の同化ステロイド使用（ドーピングの有無を問わず）により性腺機能が回復不能となった場合、WADAは「禁止物質の過去の使用に起因する必要性」としてTUEを拒否する方針 | **高** |

### 具体的シナリオ

> **例**: 52歳のマスターズ陸上選手。倦怠感、性機能低下、骨密度減少を訴えて内分泌科を受診。朝空腹時テストステロン値が2回とも220ng/dLで、Endocrine Societyガイドラインに基づきLOH（Late-Onset Hypogonadism）と診断。生活習慣改善（減量・運動）を6ヶ月試みるも改善なし。内分泌科医はTRT適応と判断するが、WADAガイドラインでは機能的性腺機能低下症に該当しTUE取得は不可。アスリートは「治療を受けるか競技を続けるか」の選択を強いられる。

## 5. 糖質コルチコイド（グルココルチコイド）

### WADAの現行規定（2022年改訂以降）

| 投与経路 | ステータス |
|---|---|
| 経口・注射（全種類）・直腸 | **競技会時禁止**（S9） |
| 吸入・経鼻・外用（皮膚）・点眼 | **許可**（TUE不要） |
| 競技会外 | 全経路で**許可**（ただしウォッシュアウト期間に注意） |

### ウォッシュアウト期間の問題

WADAは2022年以降、GC注射（関節内注射を含む）を競技会時禁止に追加し、ウォッシュアウトテーブルを導入：

| 投与経路 | グルココルチコイド | ウォッシュアウト期間 |
|---|---|---|
| 経口 | ほとんどのGC | 3日 |
| 経口 | トリアムシノロン | 10日 |
| 筋肉内注射 | ベタメタゾン・デキサメタゾン等 | 5日 |
| 筋肉内注射 | トリアムシノロンアセトニド | 60日 |
| 関節内注射 | ほとんどのGC | 3日 |
| 関節内注射 | トリアムシノロンアセトニド | 10日 |

### 最新臨床ガイドラインとの乖離

| 項目 | ギャップの内容 | リスク評価 |
|---|---|---|
| **関節内注射の禁止** | 2022年以前は競技会時でも関節内GC注射は許可されていた。現在は禁止。スポーツ医学では関節炎・腱炎に対する局所GC注射は標準治療。シーズン中の治療タイミングが大幅に制限される | **高** |
| **持続放出型製剤のリスク** | 2026年の修正で「持続放出型製剤はウォッシュアウト期間を超えてGCが検出される可能性がある」と明記。トリアムシノロンアセトニドの筋注は60日のウォッシュアウトが必要で、シーズン中の使用は事実上不可能 | **高** |
| **急性増悪への対応** | 喘息急性増悪、炎症性腸疾患の再燃、アレルギー反応等で経口GC短期投与が必要な場合、競技会直前であればTUE申請が必要。緊急時は遡及的TUEが可能だが、手続きの不確実性がある | **中** |
| **IBD（炎症性腸疾患）** | クローン病・潰瘍性大腸炎の再燃時に経口プレドニゾロンは標準的寛解導入療法。WADA TUEガイドライン（IBD）は存在するが、競技スケジュールとの両立が困難 | **中** |

## 6. 心血管疾患（利尿薬・β遮断薬）

### WADAの現行規定

| 物質 | ステータス |
|---|---|
| **全利尿薬** | **常時禁止**（S5: 利尿薬とマスキング剤）- 体重操作やドーピング隠蔽に使用される可能性 |
| **β遮断薬** | **特定競技で競技会時禁止**（射撃、アーチェリー、自動車、ゴルフ、ビリヤード等） |

### 最新心不全・高血圧ガイドラインとの乖離

#### 心不全

2025年 ESC/HFA/HFAIガイドラインでは心不全の4本柱治療（Fantastic Four）として：
1. ACE阻害薬/ARNi
2. β遮断薬
3. MRA（ミネラロコルチコイド受容体拮抗薬） - **スピロノラクトン等は利尿薬としてWADA禁止**
4. **SGLT2阻害薬**（ダパグリフロジン、エンパグリフロジン） - 禁止対象外

加えてループ利尿薬（フロセミド等）はうっ血管理に必須。

#### 高血圧

ESC/ESH 2023高血圧ガイドラインでは：
- 第一選択：ACE阻害薬/ARB、Ca拮抗薬、サイアザイド系利尿薬
- **サイアザイド系利尿薬はWADA禁止**

| 項目 | ギャップの内容 | リスク評価 |
|---|---|---|
| **利尿薬の一律禁止** | 利尿薬はマスキング剤としての懸念から禁止されているが、心不全や高血圧では臨床的に必要不可欠。代替薬（ACE阻害薬、Ca拮抗薬等）は存在するが、治療抵抗性高血圧ではサイアザイド系利尿薬が必要な場合がある | **高** |
| **MRA（スピロノラクトン）の禁止** | スピロノラクトン/エプレレノンは心不全治療の標準4本柱の一つ。WADAではS5利尿薬カテゴリーで禁止。心不全アスリートは標準治療の一角を欠いた治療を受けるか、TUEを取得する必要がある | **高** |
| **β遮断薬の精密競技での制限** | 射撃やアーチェリーの選手が心房細動や高血圧でβ遮断薬を処方された場合、CAS判例（2009、2013年）に基づきTUEが拒否される可能性がある。「パフォーマンス向上効果がないことの証明」が求められる | **非常に高** |

### 具体的シナリオ

> **例**: 45歳のマスターズ射撃選手が発作性心房細動と診断。循環器内科ガイドラインではレートコントロールにβ遮断薬（ビソプロロール）が第一選択。しかし射撃ではβ遮断薬が競技会時禁止であり、CAS判例からTUE取得も困難。Ca拮抗薬（ベラパミル）への変更は可能だが、心拍コントロールが不十分な場合がある。

## 7. PCOS（多嚢胞性卵巣症候群）・女性不妊

### WADAの現行規定

| 物質 | ステータス |
|---|---|
| クロミフェン | **常時禁止**（S4: 抗エストロゲン） |
| レトロゾール | **常時禁止**（S4: アロマターゼ阻害薬） |
| スピロノラクトン | **常時禁止**（S5: 利尿薬） |

### PCOS国際ガイドライン（2023年改訂）との乖離

| 項目 | ギャップの内容 | リスク評価 |
|---|---|---|
| **排卵誘発の第一選択薬** | PCOSによる不妊症に対する排卵誘発の第一選択薬はレトロゾール（国際PCOS/WHO推奨）。クロミフェンは第二選択。両剤ともWADA常時禁止であり、TUE必須 | **高** |
| **多毛症治療** | PCOS関連多毛症にスピロノラクトンが推奨されるが、WADA利尿薬カテゴリーで常時禁止。TUE必須 | **中** |
| **妊娠希望時のタイミング** | 排卵誘発は月経周期に合わせた限定的使用だが、WADAの「常時禁止」カテゴリーでは競技会外でもTUEが必要。妊活中のアスリートの競技継続を困難にする | **高** |
| **女性アスリートの生殖権** | TUE取得のためにプライベートな生殖医療情報をスポーツ機関に開示する必要がある。プライバシーの観点から懸念がある | **中** |

## 8. 横断的課題と提言

### 8.1 構造的問題

#### A. ガイドライン更新の時間差（タイムラグ）

WADA TUE Physician Guidelinesは「毎年レビューされる」と記載されているが、実際の改訂頻度にはばらつきがある：

| ガイドライン | 最新版 | 参照する臨床ガイドライン |
|---|---|---|
| 糖尿病 | Version 5.1（2023年10月） | ADA Standards of Care 2025との乖離 |
| ADHD | Version 8.0（2026年1月） | 比較的最新だが、実臨床の変化に追いつけていない部分あり |
| 男性性腺機能低下症 | 2025年12月更新 | Endocrine Society 2018ガイドラインは反映されているが解釈が異なる |
| 喘息 | Version 9.3（2026年1月） | GINA 2025のTrack 1推奨と整合性に課題 |
| 心血管 | Version 4.0（2023年10月） | ESC 2023/HFA 2025の新規推奨との乖離 |

#### B. TUE制度の地域間格差

- TUE審査の厳格さは各NADO（国内アンチ・ドーピング機構）やIF（国際競技連盟）のTUE委員会によって異なる
- 医療アクセスの国際格差（途上国のアスリートは専門医の診断やTUE申請書類の準備が困難）
- 言語の壁（TUEガイドラインは主に英語・仏語で提供）

#### C. 「パフォーマンス向上なし」の証明困難

TUE承認基準の一つである「禁止物質の使用がパフォーマンスを正常な健康状態への回復以上に向上させないこと」の判定は本質的に困難：
- テストステロン補充が「生理的範囲内」であってもパフォーマンスに影響する可能性
- 刺激薬がADHDの認知機能を「正常に戻す」のか「正常以上に向上させる」のかの境界が不明確
- β2作動薬の吸入が気管支拡張以上の全身的効果（筋力増強等）を持つ可能性

### 8.2 提言

| 番号 | 提言 | 優先度 |
|---|---|---|
| 1 | **TUEガイドラインの迅速な更新メカニズムの確立**: 主要臨床ガイドライン（GINA、ADA、Endocrine Society等）の改訂から12ヶ月以内にWADA TUEガイドラインも対応する仕組みの構築 | 高 |
| 2 | **機能的性腺機能低下症への対応見直し**: マスターズアスリートの増加を踏まえ、LOHに対するTRT TUEの条件付き承認枠の検討 | 高 |
| 3 | **GLP-1受容体作動薬の禁止判断に際しての影響評価**: 禁止表追加前に、糖尿病・肥満症アスリートへの治療アクセスへの影響を評価するプロセスの確立 | 高 |
| 4 | **喘息MART療法のための用量閾値見直し**: GINA Track 1のMARTレジメンに対応したホルモテロール用量上限の再検討 | 中 |
| 5 | **ADHDの「競技会時のみ禁止」分類の再評価**: 長時間作用型刺激薬の薬物動態を考慮し、競技日の服薬中断が非現実的であることを反映した制度設計 | 中 |
| 6 | **利尿薬禁止の差別化**: マスキング剤としてのリスクが低い利尿薬（MRA等）について、心不全治療目的の簡易TUEプロセスの検討 | 中 |
| 7 | **TUE申請の簡素化・デジタル化**: ADAMSシステムの改善、多言語対応の強化、申請書類の標準化によるアスリートの負担軽減 | 中 |
| 8 | **臨床医向けスポーツ・アンチドーピング教育の強化**: 処方医がWADA禁止表を意識した治療選択ができるよう、医学教育への統合 | 高 |
